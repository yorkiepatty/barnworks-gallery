## 🧠 CARDINAL RULE #2: THE NEURAL INFRASTRUCTURE MANDATE

**ALL SYSTEMS MUST OPERATE IN ACCORDANCE WITH THE BRAIN HIERARCHY**

### The Brain Structure (Physiologically Organized):

```
brain/
├── 01_cortex/          # Central command - decision making, coordination
├── 02_memory/          # Short and long-term memory, learning
├── 03_reasoning/       # Logic, NLP, interpretation, understanding
├── 04_speech/          # Voice output, TTS, speech recognition
├── 05_vision/          # Visual input, gesture recognition, eye tracking
└── 06_motor/           # Action execution, response coordination
```

### Proximity Rules (Based on Human Neurophysiology):
- **Cortex is central** - all regions connect to it
- **Memory is adjacent to cortex** - working memory must be fast
- **Reasoning connects cortex to sensory** - interpretation layer
- **Sensory (vision, speech) feeds into reasoning** - input processing
- **Motor executes from cortex** - action output

### Scientific Validation:
This structure mirrors human brain organization because:
1. It's been optimized by millions of years of evolution
2. Proximity = speed = survival for our users
3. Information flow follows natural neural pathways
4. Damage to one region doesn't collapse entire system
